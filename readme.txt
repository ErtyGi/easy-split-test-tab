This plugin help you and Google Analytics to create experiments that test which version of a landing page leads better. 

== Description ==

Through the Google Experiments and Easy Split Test Tab  you can be able to create your perfect website.
A website that reaches purposes easly through the experiments.
Helps to decide which pages generate greater results in the completion of a goal (the sale of a product, sending a request etc..).

The plugin keeps track of all the experiments within your site. 

Translations included for:

* Italiano
* English


== Installation ==

This section describes how to install the plugin and get it working.

* 1 Log In as an Admin on your WordPress website.
* 2 In the menu displayed on the left, there is a "Plugins" tab. Click it.
* 3 Now click "Add New".
* 4 There, you have the buttons: "Search | Upload | Featured | Popular | Newest". Click "Upload".
* 5 Upload the easy-split-test-tab.zip file.
* 6 After the upload is finished, click Activate Plugin.



== Frequently Asked Questions ==

- First step: Create your original Page.

- Second step: Create your variation pages. 

- Third step: Create your experiment on Google Analytics. 

- Fourth step: Create your experiment on Easy Split Test Tab.
